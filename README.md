## Visualizer

This project is developed with React to help people to view algorithmic processes intuitively and entertainingly. <br/>


## Algorithms Included

- Pathfinding Category
  - BFS
  - Dijkstra
  - A* Search
  - DFS
- Sorting Category
  - Bubble Sort
  - Insertion Sort
  - Selection Sort
  - Merge Sort
  - Quick Sort
- AI Category
  - Simple Perceptron
  - Approximate Q Learning
  - Minimax(Alpha–beta pruning)
